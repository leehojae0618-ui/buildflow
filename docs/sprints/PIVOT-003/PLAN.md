# PIVOT-003 Plan

## Goal

Requirement Snapshot을 BuildFlow가 자동 구축 가능 범위로 판단할 수 있는 확장 가능한 Capability 모델로 확장한다.

## In Scope

- 공식 AutomationLevel 정의
- Component Capability와 Capability Calculator
- Consent를 CONSENT_REQUIRED Capability와 연결
- Snapshot 저장 및 Requirement Summary 시각화
- 테스트와 제품 문서 갱신

## Out of Scope

- Migration, 새 DB table, API breaking change
- Recommendation 삭제
- Build Planner, Build Engine, Marketplace 구현
